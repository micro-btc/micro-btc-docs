`micro-btc` is a minimal JavaScript/TypeScript library for Bitcoin.

## Documentation

[Read the docs](/docs/getting-started) to get started.

## Features

### Payments

- Single-Key Addresses
  - Classic Public Key Hash
  - SegWit Public Key Hash
  - Taproot Public Key
  - Taproot Public Key Hash
- Multi-Key Addresses
  - Classic Script Hash
  - SegWit Script Hash
  - Taproot Script Hash
- Single-Script Addresses
  - Addresses with Custom Scripts
- Multi-Script Addresses
  - Taproot Script Tree
  - Taproot Key & Script Tree

### Transactions

- Encoding and Decoding
- Inputs
- Outputs
- Basic transaction signing
- Signing with PSBT (Partially-Signed Bitcoin Transactions)

### Utilities

- Getting an address
- Scripts
